# Toolkit for fake data generation in Golang

This library aims at facilitating creating fake data.

Random data available:
- [x] Name
- [x] Postal address
- [x] Paragraph
- [x] Email address
- [x] User agent
- [x] Basic user (with name, email and ID)
- [x] IP address
- [x] Unique ID (UUID v4)
- [x] URL
